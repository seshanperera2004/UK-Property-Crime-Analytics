# UK Property & Crime Analytics Dashboard

## Overview
This project analyzes UK property prices and crime data using SQL Server and Power BI.

## Key Features
- Property price trends analysis (Lancashire)
- Crime hotspot analysis (Yorkshire & Humber)
- Interactive dashboards with KPIs

## Tools Used
- SQL Server (T-SQL)
- Power BI
- Data Cleaning & Modeling

## Project Structure
- Property Analysis: SQL, datasets, dashboard
- Crime Analysis: SQL, datasets, dashboard
- Videos: Dashboard walkthroughs
